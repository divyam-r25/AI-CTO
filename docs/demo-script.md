# Demo Script (4-5 minutes)

## Opening (20 sec)
"This is AI CTO, a repo-native architect and risk simulator that turns any PRD into a build plan."

## Input (40 sec)
- Open sample PRD
- Explain user, constraints, and goals
- Select simulation mode

## Run (60 sec)
- Click Generate CTO Plan
- Show architecture section first
- Highlight roadmap phases and dependencies

## WOW Moment (70 sec)
- Open risk simulation section
- Explain RPN-driven risk scoring
- Read failure narrative and pre-build change recommendations

## Cost and Decision (50 sec)
- Show tiered cost forecast
- Explain top cost driver and optimization path
- End with recommendation verdict and confidence

## Close (20 sec)
"This is not a code generator. It is a decision engine for founders and builders before they waste sprint cycles."
